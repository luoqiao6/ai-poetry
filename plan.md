计划开发一个AI诗词推荐和赏析应用，此应用分为三部分：
一、AI 诗词推荐程序（AI 诗词推荐 Agent）

在服务器上运行的后台程序，通过调用AI大模型接口，向大模型提供提示词和上下文等信息获取大模型返回的推荐诗词。这个程序应该是一个命令程序，或可以说是一个简单的命令形式的 AI Agent，通过传入提示词等参数请求AI大模型接口，然后将AI返回的内容保存到数据库中，并返回命令执行成功或失败的状态值和提示信息。

二、应用API接口
服务器上的 http API 接口，读取数据库中从AI 大模型获取的推荐内容，返回给客户端。

三、系统运行Web后台管理系统
服务端的后台管理界面，管理第一部分的命令程序的运行参数，和获取到的推荐内容，以后还可能管理客户端用户请求输入的提示词、等相关信息。


四、客户端
 客户端即应用APP，包括适配安卓、苹果等各型号手机App、平板电脑App、PC电脑App 等


 现阶段计划先实现第一部分，开发一个简单的诗词推荐命令行程序，调研大模型API接口，实现获取AI推荐诗词功能。
